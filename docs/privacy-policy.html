Python Kids 隐私政策

最后更新日期：2024年11月

概述

Python Kids（以下简称"本应用"）是一款离线 Python 编程学习应用，我们非常重视您的隐私。本隐私政策说明了我们如何收集、使用和保护您的信息。

数据收集

本应用收集以下用户数据：

1. 学习进度数据

  • 课程完成状态
  • 练习题的答题记录
  • 学习时间统计

2. 学习计划数据

  • 用户创建的学习计划
  • 每日学习任务完成情况

3. 奖励数据

  • 用户获得的奖励记录

4. 应用设置

  • 最后查看的章节
  • 其他应用偏好设置

数据存储

所有数据均存储在您的本地设备上，具体位置：

macOS：~/Library/Containers/com.pythonkids.appstore/Data/Library/Application Support/Python Kids/data/

重要说明：

  • ✅ 所有数据仅存储在您的本地设备
  • ✅ 数据不会上传到任何服务器
  • ✅ 数据不会与第三方共享
  • ✅ 数据不会用于任何商业目的

数据使用

收集的数据仅用于以下目的：

  1. 在应用内显示您的学习进度和统计信息
  2. 恢复您的学习状态（如最后查看的章节）
  3. 提供个性化的学习计划功能
  4. 显示奖励和成就记录

数据安全

  • 所有数据存储在应用沙盒容器目录中，其他应用无法访问
  • 数据以 JSON 格式存储在本地文件中
  • 我们不会对数据进行任何加密处理，因为数据仅存储在您的设备上

数据删除

您可以随时删除您的数据：

  1. 删除应用：删除应用将同时删除所有相关数据
  2. 手动删除：删除上述数据存储目录中的文件

网络访问

本应用是完全离线应用：

  • ✅ 不需要网络连接即可使用
  • ✅ 不会发送任何网络请求
  • ✅ 不会连接到任何服务器

第三方服务

本应用不集成任何第三方服务：

  • 不使用分析服务
  • 不使用广告服务
  • 不使用云存储服务
  • 不使用社交网络服务

儿童隐私

本应用适合所有年龄段用户使用，包括儿童。我们不会：

  • 收集个人身份信息（如姓名、邮箱、电话等）
  • 收集位置信息
  • 收集设备标识符
  • 进行任何形式的用户追踪

隐私政策更新

我们可能会不定期更新本隐私政策。更新后的政策将在应用内显示，并更新本页面顶部的"最后更新日期"。

联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

  • 应用内反馈：在应用内提交反馈
  • GitHub Issues：如果您是通过 GitHub 获得的源代码

法律声明

本隐私政策符合以下法律法规：

  • 欧盟《通用数据保护条例》（GDPR）
  • 美国《儿童在线隐私保护法》（COPPA）
  • 中国《个人信息保护法》

注意：由于本应用是完全离线应用，所有数据仅存储在您的设备上，不会上传到任何服务器。您的隐私受到完全保护。
