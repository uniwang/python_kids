Python Kids Privacy Policy

Last Updated: November 2024

Overview

Python Kids (hereinafter referred to as "this app") is an offline Python programming learning application. We highly value your privacy. This Privacy Policy explains how we collect, use, and protect your information.

Data Collection

This app collects the following user data:

1. Learning Progress Data

  • Course completion status
  • Practice question answer records
  • Learning time statistics

2. Study Plan Data

  • Study plans created by users
  • Daily learning task completion status

3. Reward Data

  • Reward records obtained by users

4. Application Settings

  • Last viewed chapter
  • Other application preferences

Data Storage

All data is stored locally on your device, at the following location:

macOS: ~/Library/Containers/com.pythonkids.appstore/Data/Library/Application Support/Python Kids/data/

Important Notes:

  • ✅ All data is stored only on your local device
  • ✅ Data is never uploaded to any server
  • ✅ Data is never shared with third parties
  • ✅ Data is never used for commercial purposes

Data Usage

Collected data is used solely for the following purposes:

  1. Display your learning progress and statistics within the app
  2. Restore your learning state (such as the last viewed chapter)
  3. Provide personalized study plan features
  4. Display rewards and achievement records

Data Security

  • All data is stored in the app's sandbox container directory, which other apps cannot access
  • Data is stored in JSON format in local files
  • We do not encrypt the data, as it is stored only on your device

Data Deletion

You can delete your data at any time:

  1. Delete the app: Deleting the app will also delete all related data
  2. Manual deletion: Delete files in the data storage directory mentioned above

Network Access

This app is completely offline:

  • ✅ No network connection required to use
  • ✅ Never sends any network requests
  • ✅ Never connects to any server

Third-Party Services

This app does not integrate any third-party services:

  • No analytics services
  • No advertising services
  • No cloud storage services
  • No social networking services

Children's Privacy

This app is suitable for users of all ages, including children. We do not:

  • Collect personally identifiable information (such as name, email, phone number, etc.)
  • Collect location information
  • Collect device identifiers
  • Conduct any form of user tracking

Privacy Policy Updates

We may update this Privacy Policy from time to time. Updated policies will be displayed within the app and the "Last Updated" date at the top of this page will be updated accordingly.

Contact Us

If you have any questions or suggestions regarding this Privacy Policy, please contact us through:

  • In-app feedback: Submit feedback within the app
  • GitHub Issues: If you obtained the source code through GitHub

Legal Compliance

This Privacy Policy complies with the following laws and regulations:

  • European Union General Data Protection Regulation (GDPR)
  • United States Children's Online Privacy Protection Act (COPPA)
  • China Personal Information Protection Law

Note: Since this app is completely offline, all data is stored only on your device and never uploaded to any server. Your privacy is fully protected.
